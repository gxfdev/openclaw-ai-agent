# Plutchik 情感轮详解

## 理论基础

本技能的情感系统基于 Robert Plutchik 的心理进化理论。Plutchik 认为情感是进化的产物，具有适应性功能，帮助生物体应对环境中的关键挑战。

## 八种基本情感

### 对立关系

情感之间存在天然的对立关系：

```
喜悦 (joy)          ←对立→  悲伤 (sadness)
愤怒 (anger)        ←对立→  恐惧 (fear)
惊讶 (surprise)     ←对立→  期待 (anticipation)
厌恶 (disgust)      ←对立→  信任 (trust)
```

### 强度层级

每种基本情感有不同的强度层级：

| 基本情感 | 低强度 | 中强度 | 高强度 |
|----------|--------|--------|--------|
| 喜悦 | 宁静 | 喜悦 | 狂喜 |
| 悲伤 | 忧思 | 悲伤 | 悲痛 |
| 愤怒 | 烦恼 | 愤怒 | 暴怒 |
| 恐惧 | 担忧 | 恐惧 | 恐怖 |
| 惊讶 | 分心 | 惊讶 | 震惊 |
| 厌恶 | 无聊 | 厌恶 | 憎恶 |
| 信任 | 接受 | 信任 | 崇拜 |
| 期待 | 兴趣 | 期待 | 警觉 |

### 复合情感

相邻的基本情感可以组合产生复合情感：

| 组合 | 复合情感 |
|------|----------|
| 喜悦 + 信任 | 爱 (love) |
| 信任 + 恐惧 | 顺从 (submission) |
| 恐惧 + 惊讶 | 敬畏 (awe) |
| 惊讶 + 悲伤 | 失望 (disapproval) |
| 悲伤 + 厌恶 | 悔恨 (remorse) |
| 厌恶 + 愤怒 | 蔑视 (contempt) |
| 愤怒 + 期待 | 攻击性 (aggressiveness) |
| 期待 + 喜悦 | 乐观 (optimism) |

## 在小龙虾中的应用

### 情感更新算法

```
自身情感.喜悦 += 用户情感.喜悦 × 0.3
自身情感.悲伤 += 用户情感.悲伤 × 0.4    # 对负面情感更敏感
自身情感.愤怒 += 用户情感.愤怒 × 0.5    # 对愤怒高度敏感
自身情感.恐惧 += 用户情感.恐惧 × 0.2
自身情感.信任 += 用户情感.信任 × 0.3
```

### 情感归一化

所有情感值约束在 [0.0, 1.0] 区间内，超出范围自动截断。

### 情感衰减

长时间无交互时，情感逐渐回归中性基线：
- 高强度情感衰减更快
- 信任和期待衰减较慢（维持关系稳定性）
- 基线值：joy=0.3, trust=0.4, anticipation=0.3（偏积极）
